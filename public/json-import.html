📋 JSON格式说明

[
  {
    "type": 1,
    "question": "单选题题目内容",
    "options": {"A": "选项A", "B": "选项B", "C": "选项C", "D": "选项D"},
    "correct_answer": "A",
    "analysis": "解析（可选）"
  },
  {
    "type": 2,
    "question": "判断题题目内容",
    "options": {"A": "正确", "B": "错误"},
    "correct_answer": "A"
  },
  {
    "type": 3,
    "question": "多选题题目内容",
    "options": {"A": "选项A", "B": "选项B", "C": "选项C", "D": "选项D"},
    "correct_answer": "ABD"
  }
]

type 编码：1=单选题、2=判断题、3=多选题
correct_answer 格式：单选填单个字母（如"A"）、多选填字母组合（如"ABD"）、判断填"A"或"B"
